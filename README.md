## About

Professional video editing software for filmmakers.

## Features

- Timeline editing
- Effects
- Color grading
- Audio editing
- Export options

## Installation

### From App Store
Open the **App Store** in zOS and search for "Final Cut Pro".

### Via npm
```bash
npm install @zos-apps/final-cut-pro
```

## Usage

- Import media to browser
- Drag clips to timeline
- Add effects from sidebar



## Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```
